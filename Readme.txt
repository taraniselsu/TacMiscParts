===== Features =====
* Medium SAS module -- smaller version of the SAS flight computer, with no reaction
 wheel torque. For use with normal sized rockets and space planes.
* Small SAS module -- probe-sized version of the SAS flight computer, with no
 reaction wheel torque. For use with small probes or satellites.
* Small Reaction Wheel -- probe-sized version of the Reaction Wheels. Produces
 torque but does not contain the flight computer for holding an orientation. For use
 with small probes or satellites.

===== Installation procedure =====
1) Copy everything in the GameData directory to the {KSP}/GameData directory.
